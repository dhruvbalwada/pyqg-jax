xarray Output

Convert model states into labeled xarray.Dataset objects.

This module provides state_to_dataset(), which packages the fields of a model state together with spatial and spectral coordinates into an xarray.Dataset for analysis and serialization.

This module requires the optional dependency xarray.

pyqg_jax.xarray_output.state_to_dataset(state, model)[source]

Package a model state into an xarray.Dataset.

Real-space fields are given dimensions ("lev", "y", "x") and spectral fields ("lev", "l", "k"), with matching coordinates derived from the model grid. Whichever of the fields q, qh, u, v, p, ph are present on state are included (so both a PseudoSpectralState and a FullPseudoSpectralState are accepted).

Added in version 0.9.0.

Parameters:

  • state (PseudoSpectralState or FullPseudoSpectralState) – The state to convert. This function operates on a single time step; to build a dataset for a trajectory, convert snapshots and combine them (for example with xarray.concat()).

  • model – The model that produced state, used for the grid coordinates and model parameters (stored as dataset attributes).

Returns:

The state fields with labeled spatial/spectral coordinates.

Return type:

xarray.Dataset
